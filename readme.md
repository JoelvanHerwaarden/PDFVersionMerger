Nobleo Bouw en Infra | PDF Vergelijker 📄🤖
Een lichtgewicht, browser-gebaseerde tool om twee PDF-documenten zij-aan-zij samen te voegen. Ontworpen voor engineers en projectmanagers bij Nobleo Bouw en Infra om revisies, berekeningen en tekeningen visueel te valideren.

🌐 Live Applicatie
Je kunt de tool direct gebruiken via:

https://joelvanherwaarden.github.io/PDFVersionMerger/

🚀 Kenmerken
Zij-aan-zij Vergelijking: Voegt twee PDF's samen in een Landscape-formaat voor directe visuele analyse.

100% Privacy: De verwerking vindt volledig plaats in de browser (client-side). Documenten worden nooit geüpload naar een server.

Dark Mode Support: Schakel eenvoudig tussen licht en donker thema voor optimaal werkcomfort.

Dynamische Schaling: Houdt rekening met verschillende paginalengtes tussen documenten.

Vibe-coded: Gebouwd met een focus op moderne UI/UX principes.

🛠️ Technologie
Deze tool maakt gebruik van de volgende technologieën:

JavaScript (Vanilla): Voor de logica en UI-interacties.

pdf-lib: Een krachtige bibliotheek voor het modificeren en genereren van PDF-documenten in de browser.

CSS Custom Properties: Voor dynamische thema-wisselingen.

📖 Hoe het werkt
Referentie Document: Selecteer het originele bestand (verschijnt links).

Wijziging / Variant: Selecteer het aangepaste bestand (verschijnt rechts).

Genereer: Klik op de knop om de vergelijking te starten.

Download: De samengevoegde PDF wordt automatisch gedownload als Nobleo_Bouw_Infra_Vergelijking.pdf.

🔒 Beveiliging & Privacy
Omdat deze applicatie volledig in de sandbox van je browser draait:

Blijven vertrouwelijke bedrijfsgegevens lokaal op je machine.

Is er geen internetverbinding nodig nadat de pagina is geladen.

Worden er geen cookies of tracking-scripts gebruikt voor documentgegevens.
